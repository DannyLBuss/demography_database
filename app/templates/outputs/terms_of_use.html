User Agreement

Please Read

The COMPADRE Plant Matrix Database and COMADRE Animal Matrix Database provided via this portal are open-access data and are free of charge. We request that in all published work and presentations, the user acknowledges the COMPADRE Plant Matrix Database and/or the COMADRE Animal Matrix Database as either the source or the intermediary of the data. In the interest of reproducibility by the scientific community, we suggest the following citation format in outputs:

  • COMPADRE Plant Matrix Database. University of Exeter Cornwall Campus (United Kingdom). Available at demography.ex.ac.uk. (data downloaded on [date], version X.X.X).

  • COMADRE Animal Matrix Database. Max Planck Institute for Demographic Research (Germany). Available at www.comadre-db.org. (data downloaded on [date] , version X.X.X).


  • The version number of the database for COMPADRE and/or COMADRE is indicated within their respective Rdata objects (e.g compadre$version).


  • If the user chooses another citation format, the full name of the database(s) (COMPADRE Plant Matrix Database and/or COMADRE Animal Matrix Database), the full name of the institutional sponsors (University of Exeter, Max Planck Institute for Demographic Research), the version used, and our internet addresses (demography.ex.ac.uk) must be explicitly mentioned.

If you have further questions about citation procedures, please contact compadrecontact@gmail.com.
