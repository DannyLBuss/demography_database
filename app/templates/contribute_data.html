How to Contribute Data

Submitting data

Currently, our compadrino team input all the data into the database. If you have data that you want to contribute to this open access database you can do the following:

  1. If your data is attached to a scientific publication and this publication cannot be found on the database (check here) then please submit you publication data into the database here.
  2. Email data or a copy of the transcript to compadre@gmail.com and reference the publication ID or DOI/ISBN

Data requirements and guidelines

  • Present every matrix you have constructed as a grid of numbers, rather than raw demographic data.
  • If possible, present three submatrices that represent the fecundity, survival and clonal transitions of the matrix.
  • meta data etc.
